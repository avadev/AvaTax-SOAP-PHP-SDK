Class Details

A Jurisdiction Type describes the jurisdiction for which a particular tax is applied to a document.

A Jurisdiction Type describes the jurisdiction for which a particular tax is applied to a document. Jurisdiction is determined by the GetTaxRequest#getDestinationAddress of a GetTaxRequest. Multiple jurisdictions might be applied to a single Line during a tax calcuation. Details are available in the TaxDetail of the GetTaxResult.
